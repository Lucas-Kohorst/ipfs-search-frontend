IPFS Search

API for the Interplanetary Filesystem

Overview

Use this API to make request to ipfs-search. You can make requests for specific filetypes, only directories or only files, time last seen, metadata, and get all items that are linked to a parent hash.

Error Codes

If a query does not return any results you can except a message with a total of 0

{
    "total": 0,
    "max_score": null,
    "hits": [],
    "page_size": 15,
    "page_count": 0
}

Rate limit

No rate limit

GET v1/search?q=ipfs

Basic Search Query

Params

Parameter Example Value
q ipfs

Example Request
https://api.ipfs-search.com/v1/search?q=ipfs

curl --location --request GET "https://api.ipfs-search.com/v1/search?q=water" \
--data ""

Parameter Example Value
